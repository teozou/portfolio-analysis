# Portfolio Analysis

Quantitative analysis of a financial portfolio built with Python.

## What this project covers
- Data collection via yfinance (AAPL, MSFT, GOOGL, AMZN, JPM)
- KMeans clustering for diversification
- Risk metrics : VaR (historical, gaussian, Monte Carlo), CVaR, backtesting
- Markowitz optimization with Ledoit-Wolf shrinkage
- Walk-forward backtest
- Random Matrix Theory (noise filtering)
- Benchmark comparison (CAC 40, S&P 500, MSCI World)

## Stack
Python · Pandas · NumPy · Scikit-learn · Scipy · Matplotlib · Seaborn · yfinance

## How to run
pip install -r requirements.txt

Then open the notebook in Jupyter.
